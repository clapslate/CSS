CPU Scheduler Simulator (CSS)
Author: Mohit Gvalani
Email: mohitindubai@gmail.com

This Java application is a simulator that has been tested on WindowsXP with JRE6 installed. It simulates scheduling and execution of processes on a CPU.
It contains 6 scheduling algorithms.
-FCFS
-Round Robin
-Shortest Process Next
-Shortest Remaining Time
-Non-Preemptive Priority
-Preemptive Priority
Unit for time is 1 sec. To remove a process from queue, click on the process and then click on "dump".
